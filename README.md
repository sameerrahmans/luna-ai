# Luna AI

Luna AI is a modern AI-powered chatbot built with Next.js and designed with a clean, minimal, industry-standard user interface. It delivers fast and intelligent conversational responses with a smooth, responsive experience across desktop and mobile devices.

---

## Features

- Modern AI chatbot interface
- Clean and minimal design
- Typing animation for AI responses
- Local storage chat history (no login required)
- Mobile responsive layout
- Keyboard-safe input behavior
- Lightweight and fast architecture

---

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Groq API (LLM backend)
- Lucide Icons
- Local Storage for chat persistence
